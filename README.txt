ABOUT
-----
This is the personal portfolio website of Kazi Zehad Mostofa, a PhD researcher at Clarkson University
specializing in energy systems optimization. The site showcases research projects, peer-reviewed
publications, work experience, technical skills, and conference presentations across the fields of
deterministic and probabilistic load forecasting, renewable energy integration, microgrid control,
and sustainable building energy design.

RUNNING LOCALLY
---------------
No build step required. To preview the site locally:

  1. Clone or download the repository:
       git clone https://github.com/Mostofa-Zehad/Mostofa-Zehad.github.io.git

  2. Open the folder and launch a local server. Using Python:
       cd Mostofa-Zehad.github.io
       python3 -m http.server 8080

  3. Open your browser and go to:
       http://localhost:8080

  Alternatively, open index.html directly in a browser. Note that some assets
  (Google Fonts, Font Awesome) require an internet connection to load correctly.
