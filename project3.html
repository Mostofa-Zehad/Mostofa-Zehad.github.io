Machine Learning-Enhanced All-PV Blended Energy Systems for Energy-Efficient Sustainable Buildings

This project explores a fully photovoltaic (PV)-powered building energy model incorporating rooftop, window-integrated, and semitransparent PV systems. The goal is to optimize energy self-sufficiency while reducing carbon emissions through intelligent simulation, machine learning, and cloud-integrated monitoring.

1. Problem Statement & Motivation

Modern buildings face challenges integrating renewable energy without compromising design or functionality. Traditional PV solutions lack flexibility and holistic energy insights. This project proposes a scalable architecture utilizing multiple PV surfaces and machine learning to boost efficiency, adaptability, and real-time analytics.

2. System Architecture

The architecture includes rooftop PV modules, semitransparent BIPV windows, and conventional window-integrated PV systems. The layout facilitates full solar coverage for both residential and institutional buildings.

3. Machine Learning & Simulation Framework

Using HOMER Pro and Python-based ML regression models, the project simulates optimal sizing and forecasting. Features such as solar irradiance, tilt angles, building orientations, and hourly consumption profiles were used for predictive modeling.

4. Real-Time Monitoring via ThingSpeak IoT

ThingSpeak dashboard was created to visualize real-time data streams including solar output, power consumption, and system alerts. This allows cloud-based fault diagnostics and performance tracking from anywhere.

5. Performance Simulation & Comparison

Energy yields were simulated for 7 cities with different PV panel combinations. The city comparison below reveals significant gains from semitransparent PV and optimized configurations.

The following simulation layout illustrates hybrid AC/DC connection, bi-directional metering, and net metering benefits. It highlights system topology, load management, and inverter design.

7. Key Highlights & Achievements

  • Achieved up to 78% self-sufficiency across locations using multi-surface PV models
  • Enabled intelligent scheduling and forecasting using ML-driven predictions
  • Integrated ThingSpeak for dynamic dashboard reporting
  • Demonstrated carbon reduction through city-wide scenario modeling

8. Conclusion

This research promotes a future-ready sustainable building design leveraging diverse PV structures and intelligent tools. The project bridges the gap between renewable integration and practical building design while ensuring user control and transparency.
