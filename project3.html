Machine Learning-Enhanced All-PV Energy System

This project introduces a hybrid solar power generation approach utilizing rooftop PV, solar windows, and semitransparent photovoltaic glass. Integrated with smart grid technology and machine learning optimization, the system achieves high energy self-sufficiency for residential buildings across various global cities.

Key Achievements:

  • Achieved up to 78% energy self-sufficiency using BIPV systems.
  • Optimized PV performance using ML algorithms for real-time decision support.
  • Reduced carbon emissions and enhanced grid stability across various climates.
  • Demonstrated IoT connectivity with real-time monitoring via ThingSpeak dashboard.

Conclusion

This research demonstrates that a well-integrated all-PV building system — empowered by intelligent control — can significantly reduce dependency on the grid while maintaining architectural integrity and environmental performance. It sets the foundation for scalable deployment of intelligent energy-efficient infrastructure.
